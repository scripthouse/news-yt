## Style Guide

### Colors

**Primary**

- Soft orange: hsl(35, 77%, 62%)
- Soft red: hsl(5, 85%, 63%)

**Neutral**

- Off-white: hsl(36, 100%, 99%)
- Grayish blue: hsl(233, 8%, 79%)
- Dark grayish blue: hsl(236, 13%, 42%)
- Very dark blue: hsl(240, 100%, 5%)

**Typography**

- Font size (paragraph): 15px
- Weights: 400, 700, 800

**Layout Size**

- Mobile: 375px
- Desktop: 1440px
